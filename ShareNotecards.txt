Share notecards
Context - user is logged in and on the notecard page

Primary Flow
1 - user selects specific sets of notecards
2 - user selects button to share notecards
3 - system sends notecards to another user

Alternative Flow(s)
SYSTEM FAILS TO SHARE NOTECARDS
a4 - system says failed to share notecards
a5 - system prompts user if they'd still like to share
a6 - user clicks yes button
a7 - system shares notecards again
